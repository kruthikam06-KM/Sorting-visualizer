# Sorting-visualizer
A terminal-based Sorting Visualizer built in C that demonstrates the working of sorting algorithms like Bubble Sort, Insertion Sort,  Selection Sort in real-time using dynamic console rendering.The project uses step by step algorithm execution, interactive controls, and Windows console APIs to simulate graphical visualization within the terminal.
**Features
- Real-time sorting visualization
- Bubble, Insertion, Selection Sort
- Pause/Resume functionality
- Adjustable speed
- Interactive keyboard controls
** Controls
1 - Bubble Sort
2 - Insertion Sort
3 - Selection Sort
S - Start
P - Pause
R - Reset
N - New Array
+/- - Speed control
Q - Quit
